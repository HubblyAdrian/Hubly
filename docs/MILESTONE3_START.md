# Milestone 3 — Hubly v3 Business OS

**Start here:** [`HUBLY_V3_BUSINESS_OS.md`](./HUBLY_V3_BUSINESS_OS.md)

## What changed

| Before (M2) | After (v3) |
|-------------|------------|
| AI onboarding *was* the product | AI **builds**; Hubly **operates** |
| Reveal = reports / confidence / capsule | Reveal = visual ready cards |
| Login → Business Home / Daily | Login → **existing dashboard = Home** |
| Website = URL + Copy | Website = **large preview hero** |
| Many AI recommendations | **One** recommendation |
| Launch / certificates / morning brief primary | Archived behind Create exit |

## Modes

1. **Create** — Landing → conversation → live build → visual Reveal → Continue  
2. **Operate** — OS chassis (CRM, Jobs, Revenue, Calendar, Website, …) with Ask Hubly above every feature

## Gate

```bash
npm run milestone3   # or npm run check:m3-v3-os
```

## Chassis rule

Do not replace CRM, Jobs, Revenue, Customers, Quotes, Calendar, Website, Messaging, Marketplace, or Payments. Those stay. AI sits above them.
